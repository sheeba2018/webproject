Study Question Set 2

Q - What is an attribute?
A - Attributes are added to an element to provide the browser with more information about how the element should appear or behave.
Q - What is an element?
A - An element is a structural component (such as a header) or a desired behavior (such as a line break).
Q - What is a division element?
A -A division element denotes areas of a document, they basically make up a web page.
Q - What is meant by "self-terminating"?
A - Line breaks do as their name says...break a line to the next line.
Q - What does XHTML stand for?
A - Extensible Hypertext Markup Language

Study Question Set 3

Q -What does CSS stand for?
A -CSS (Cascading Style Sheets)
Q - What is a selector?
A - In CSS, selectors are patterns used to select the element(s) you want to style.
Q - What is a declaration?
A - There can be multiple declarations for a rule this is known as a declaration block.
Q - What is a property?
A - The declaration breaks down further, into property: value pairs, Property: font-size
Q - What is a value?
A - The declaration breaks down further, into property: value pairs, Value: 16px
Q - What is a contextual selector?
A - A descendant selector (formally known as contextual) is one that matches a specific pattern of nested elements.
Q - What does is the rel attribute of the linking element do?
A - Specifies the linked document's relation to the current document.

Study Question Set 4

Q -What is class?
A -A class is a type of CSS selector that can be applied to any element inside the body element
Q - What is an id?
A - An id is a unique identifier that is specified using the id attribute, which is also a core attribute
Q -What is a Pseudo-class?
A - A pseudo-class is used to define a special state of an element.
Q -What is the cascade?
A -The cascade is a set of guidelines governing how various rules (and style sheets) interact.
Q - What is specificity?
A Single elements have a specificity of 1 (regardless of the element).
Q - What is user style sheet?
A - User stylesheets are an exciting feature of Cascading Style Sheets (CSS)
Q - What is padding?
A - TThe padding area extends to the border surrounding the padding.
Q - What is margin?
A - A margin is the space outside something, whereas padding is the space inside something.

Study Question Set 5

Q -What is the anchor element used for?
A -The anchor element, coded as , serves as the foundation for the Web. It allows you to link pages together and create within page links.
Q - What is an absolute link?
A - Absolute links give the complete address to the document online. You must include the http:// for these to work properly.
Q -What is an relative link?
A - Relative links use a partial address, based on the relationship between the current document and the document to load.
Q - What is an inline element?
A - Inline elements are elements that surround text content
Q - What is the font family?
A - The font-family property specifies what fonts should be applied to content.
Q - Why use Google fonts?
A - Google has solved the web font limitation problem with Google web fonts
